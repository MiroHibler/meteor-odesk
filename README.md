# meteor-odesk

Meteor oDesk Login OAuth Flow.


## Dependencies

 * Meteor v0.7.1+


## TL;DR;

_meteor-odesk_ enables login with [oDesk](http://odesk.com) account.

## Installation

Install using [Meteorite](https://github.com/oortcloud/meteorite) - Installer & smart package manager for Meteor:

```sh
$ mrt add odesk
```
